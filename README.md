# tf-cartpole-study

The goal is to create a minimal yet successful training setup for a cartpole agent employing only the `tf-agents` library.

### Setup

Follow the instructions here: https://developer.apple.com/metal/tensorflow-plugin/

### Run

```shell
$ python main.py
```
